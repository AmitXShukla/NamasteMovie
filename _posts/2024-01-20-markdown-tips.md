Here are some useful Markdown tips to make your blog posts look great!

## Basic Formatting

| Syntax | Result |
|--------|--------|
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `~~strikethrough~~` | ~~strikethrough~~ |
| `` `code` `` | `code` |

## Adding Images

```markdown
![Alt text](/path/to/image.jpg)
```

You can add images to your site by placing them in an `assets/images/` folder.

## Creating Links

- External: `[Google](https://google.com)`
- Internal: `[About Page](/about)`
- With title: `[Link](url "Title on hover")`

## Task Lists

Jekyll supports GitHub-flavored Markdown task lists:

- [x] Create Jekyll site
- [x] Write first blog post
- [ ] Customize the theme
- [ ] Add more content

Markdown makes writing content enjoyable. Just focus on your ideas, and Jekyll handles the rest!
